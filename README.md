# Appliance Energy Consumption Classification

This project predicts appliance energy consumption using machine learning.  
The problem is converted into a binary classification task: **Low Energy** and **High Energy**.

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Simple Neural Network

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow/Keras
- Matplotlib
- Seaborn

## Project Workflow
1. Data cleaning
2. Feature scaling
3. Exploratory Data Analysis (EDA)
4. Model training
5. Model evaluation
6. Model comparison

## Evaluation Metrics
- Accuracy
- F1-score
- Confusion Matrix
- ROC-AUC

## What I Learned
This project helped me understand the full machine learning pipeline, from data preprocessing to model evaluation.
